# SHIP Interactive Tools

A collection of lightweight, static tools designed to support SHIP counselors in completing key tasks, including PDEO cost calculations.

## Overview

- Built using HTML, CSS, and JavaScript  
- Client-side only (no backend or data storage)  
- Designed for ease of use and low cognitive load  
- Mobile-friendly and accessible for quick use in the field  

## PDEO Calculators

The PDEO Calculators support users in calculating the **Original PDP/MA-PD Cost** for specific enrollment scenarios, following the STARS Manual.

- New to Medicare PDP/MA-PD (Consistent Non-30-Day Refill Schedule)  
- New to Medicare PDP/MA-PD (Mixed Non-30-Day Refill Schedule)  
- New to Medicare PDP/MA-PD (this year) and Switching PDP/MA-PD (next year) – 30-Day Refill Schedule  

## Technical Notes

- Static HTML/CSS/JavaScript applications  
- No backend or server-side processing required  
- No data is stored or transmitted  
- Suitable for hosting via GitHub Pages or similar static hosting platforms
