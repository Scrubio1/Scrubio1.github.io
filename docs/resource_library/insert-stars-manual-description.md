How to insert the html STARS Manual description 

Click the clipboard icon in the component to copy the code
Go to the Resource Library and Select the STARS Manual
Select Edit
In the Description box, select all text within the description and press delete (I've backed up your current description )
Select HTML
Press CTRL+V to paste in the HTML Code
Select Design
Check the update date and other information is correct
Select Upload
