
# How to Insert an HTML Block into a Resource Library Page

Click the clipboard icon in the component above to copy the code
Go to the Resource Library and Select the item you wish to edit in the Resource Library
Select Edit
In the Description box, place your cursor after the existing text and press Enter twice
Select HTML
Press CTRL+V to paste in the HTML Code
If <div style="background-color:#075190; color:#ffffff; padding:20px; border-radius:4px; font-family:Arial, sans-serif; line-height:1.5;"> appears on the same line as other text, place your cursor before this text and press Enter
Select Design
Check the update date and other information is correct
Select Upload
