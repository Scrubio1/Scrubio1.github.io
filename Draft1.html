SHIP Interactive User Tools

PDEO Calculators

Simple online calculators for finding PDP/MA-PD costs for several PDEO scenarios.

STARS Login Troubleshooter

A straight-forward guide to identify and learn to resolve common STARS login and access issues.

An interactive reference showing each STARS user role’s capabilities in the data system.
